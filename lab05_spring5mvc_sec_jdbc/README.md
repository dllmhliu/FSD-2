# Course App (spring mvc)

![lab05 screenshot](./login.jpg)

## Tech stack

1. [x] Spring MVC 5
2. [x] Spring Security 5
3. [x] Spring JDBC
4. [x] JSP/EL/JSTL
5. [x] Bootstrap 4.3
6. [x] Tomcat 9
7. [x] H2 In-Memory Database

## How to run

1. Start up tomcat 9 server
2. Open in browser: http://localhost:8080/course-app/
